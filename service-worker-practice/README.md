# service worker 实战代码

## 项目运行

```bash
yarn install

yarn add register-service-worker
```

## 基本的 register-service-worker

1. 所有构建文件的预缓存
2. 离线加载应用程序的能力
3. 在应用程序中使用 `service worker` 中的注册事件
